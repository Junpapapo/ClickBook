자주 묻는 질문 및 Q&A

ClickBook 프리미엄 익스텐션 사용 중 발생하는 궁금증과 해결책을 모았습니다.

Q 새 탭으로 설정했을 때 크롬에서 "Google로 되돌리시겠습니까?"라는 알림이 떠요. 제어할 수 없나요?

아니요, 해당 팝업은 확장 프로그램 개발자가 프로그램 코드로 강제 제어하거나 우회할 수 없습니다.

이 알림창은 크롬 브라우저가 사용자 모르게 검색엔진이나 새 탭 페이지를 조작하는 악성 소프트웨어(브라우저 하이재커)로부터 사용자를 보호하기 위해 띄우는 크롬 자체의 하드코딩된 보안 장치입니다.

정상적으로 ClickBook을 새 탭 페이지로 사용하기 위해서는 최초 설치 후 첫 새 탭을 열었을 때 반드시 [유지 (Keep changes)] 버튼을 직접 클릭해 주셔야 합니다.

Q 실수로 "이전 설정으로 되돌리기"를 눌렀더니 새 탭이나 대시보드가 하얗게 비어 있거나 꼬여서 안 열려요.

실수로 [이전 설정으로 되돌리기]를 클릭하면 크롬 브라우저가 ClickBook의 새 탭 주소 우회 권한을 일방적으로 박탈합니다. 이때 ClickBook 내부 설정과 브라우저 차단 권한의 동기가 깨지며 화면이 꼬이거나 먹통이 될 수 있습니다.

이럴 때는 아래의 조치에 따라 브라우저 보안 설정을 리셋해 주면 즉시 해결됩니다.

가장 빠르고 확실한 복구 절차:
  1. 크롬 주소창에 chrome://extensions/를 입력하여 확장 프로그램 관리 화면으로 이동합니다.
  2. ClickBook 카드의 오른쪽 아래 파란색 **활성화 스위치(토글)를 한 번 껐다가(OFF) 다시 켭니다(ON)**.
  3. 이제 빈 새 탭을 다시 열어봅니다. 크롬이 보안 알림 팝업을 새로 띄워줄 것입니다.
  4. 이때 반드시 **[유지 (Keep changes)]** 버튼을 클릭하면 즉시 모든 기능이 정상 복구됩니다.
Q 크롬 북마크를 가져오거나 대량 등록할 때 "등록 중..." 상태로 오랫동안 멈춰 있고 느립니다.

사용자 PC 브라우저에 **크롬 내장 로컬 AI(Gemini Nano)** 환경이 완전히 구성되어 있지 않을 때 발생하던 임포트 동결 병목 버그이며, **1.2.5 업데이트를 통해 완벽히 해결**되었습니다.

현재 ClickBook은 임포트 및 등록 시 내장 로컬 AI 가용 여부를 초고속(1ms 미만)으로 실시간 사전 진단합니다. AI를 활용할 수 없는 환경일 경우 세션 생성을 전면 우회하고 초고속 패턴 매칭 카테고리 분류기로 자동 낙하(Fallback) 처리하여 정지 없이 신속한 임포트를 지원합니다.

가져온 북마크의 AI 요약과 상세 태그는 백그라운드 스레드에서 비동기로 천천히 생성되므로, 대시보드 조작과 동작에는 일절 지장을 주지 않습니다.

Q 스마트 절전 모드(Tab Suspender)가 작동할 때 수면 상태로 가선 안 되는 중요 탭들이 차단되나요? Premium

걱정하지 않으셔도 됩니다. ClickBook의 스마트 절전 알고리즘은 사용자의 웹 사용성을 최우선으로 지킵니다. 아래 탭들은 사용자의 작업 보호를 위해 **절대로 강제 절전(수면) 상태로 전환되지 않고 보존**됩니다.

  • 오디오 재생 탭: 유튜브, 뮤직 스트리밍 사이트 등 백그라운드에서 음악이나 영상 음성이 송출 중인 탭.
  • 고정(Pinned) 탭: 브라우저 탭 바 왼쪽에 고정된 상시 활성화 탭.
  • 현재 활성 탭: 사용자가 지금 바라보고 조작 중인 활성화된 탭.
  • 폼 작성 중인 페이지: 인풋 입력 폼 등이 감지되는 활발한 작업 영역.
Q 보안 폴더(Secure Folder)의 자동 세션 파쇄기는 어떤 흔적들을 파쇄해 주나요? Premium

기밀 폴더나 개인 금융 도메인을 **보안 폴더**로 설정하면 해당 페이지를 볼 때 크롬 툴바 아이콘 위에 초록색 **`SEC` 배지**가 빛납니다. 탐색이 종료되고 관련 탭을 모두 닫는 순간, 오직 해당 웹 도메인(Origin)에 수집된 아래 정보들이 완전히 자동 파쇄됩니다.

  • 쿠키 (Cookies): 자동 로그인 상태 및 트래킹 세션 데이터
  • 로컬 및 세션 스토리지 (LocalStorage, SessionStorage)
  • 웹 데이터베이스: IndexedDB, WebSQL 흔적
  • 사이트 캐시 및 서비스 워커 (Service Workers, Cache)
  • 방문 기록 및 파일 잔해

사용자가 복잡하게 매번 크롬 개인정보 삭제창을 열고 쿠키를 삭제할 필요 없이, 보안 탭을 닫기만 하면 완벽히 청소됩니다.

Q 크롬 순정 탭 그룹(Tab Group)과 ClickBook 폴더는 어떻게 동기화하나요? Premium

ClickBook은 크롬 탭 그룹과의 완벽한 양방향 보존 동기화를 지원합니다.

  • 탭 그룹을 ClickBook 폴더로 저장: 확장 프로그램 아이콘(팝업)을 누르면 현재 열린 크롬 탭 그룹 목록이 컬러칩과 함께 나열됩니다. 원하는 그룹을 클릭하고 **[Save Group]**을 누르면 그룹 내부의 모든 탭이 통째로 저장된 클릭북 새 폴더가 생성됩니다.
  • ClickBook 폴더를 크롬 탭 그룹으로 복원: 대시보드 사이드바의 폴더에 마우스 호버 시 나오는 레이어 버튼을 누르거나, 폴더 내부 뷰의 헤더 우측에 위치한 **[Open as Chrome Tab Group]** 버튼을 누르면 ClickBook 폴더 이름과 지정했던 고유 컬러 상태 그대로 크롬 상단 탭바에 순정 탭 그룹으로 즉시 재생성됩니다.
Q 새 컴퓨터로 북마크 데이터를 옮기거나 전체 데이터를 백업해 두고 싶어요.

ClickBook은 오직 사용자의 컴퓨터 내부(로컬)에만 북마크를 보존하므로 수동 백업을 제공합니다.

대시보드 우측 상단의 톱니바퀴 설정을 클릭하신 뒤 데이터 관리 탭으로 이동합니다. [백업 내보내기 (JSON)]를 누르면 전체 폴더 구조, 북마크 상세 정보, 수면 설정, 보안 내역이 파일로 다운로드됩니다.

새 컴퓨터 혹은 브라우저에서 동일한 경로의 [백업 가져오기] 버튼을 통해 해당 JSON 파일을 업로드하면 1초 만에 모든 데이터가 기존 구조와 이모지까지 일치한 상태로 깔끔하게 복원됩니다.

Q 스마트 클리퍼 및 본문 기반 검색(FTS)은 어떻게 사용하며 어떤 특징이 있나요? Premium

스마트 클리퍼(Smart Clipper)는 북마크 저장 시 해당 웹페이지의 레이아웃 소음(광고, 내비게이션, 푸터 등)을 백그라운드에서 분석 및 필터링하여 순수 본문 내용만 신속하게 로컬 데이터베이스에 격리 저장합니다.

이렇게 수집된 데이터는 본문 기반 로컬 검색(FTS, Full-Text Search) 기능으로 즉시 활용됩니다:

  • 공백(스페이스) 구분 다중 검색: 검색창에 React 상태 관리와 같이 공백으로 구분된 여러 검색어를 입력하면, 해당 단어들이 모두 본문(또는 제목, 요약, 태그 등)에 포함된 항목만 실시간으로 필터링해 냅니다.
  • 오버헤드 제로: 무거운 검색 프레임워크나 외부 라이브러리 없이 순수 로컬 데이터 가공 및 인덱스 캐싱 구조로 구축되어 수천 개의 본문도 10ms 이내에 초고속 검색이 지원됩니다.
Q 오프라인 독서 모드(Offline Reader) 및 젠 리더(Zen Reader)란 무엇인가요? Premium

오프라인 독서 모드는 저장한 북마크의 본문 기사를 광고나 산만한 화면 레이아웃 없이 깔끔하고 고급스러운 책 형태로 변환해 주는 몰입형 독서 도구입니다.

  • 감각적인 테마 및 타이포그래피: 눈의 피로를 최소화하는 라이트/다크/세피아 테마를 지원하며, 영문/일어/국문 가독성에 맞춘 최적화된 서체군(Serif, Sans, Mono)과 4단계 크기 설정을 제공합니다.
  • 지능형 목차(TOC) 생성: 본문의 제목 구조(headings)를 파싱하여 좌측 목차를 자동으로 그려주며, 클릭 시 부드럽게 해당 본문 위치로 이동합니다.
  • 진척도 및 도구 모음: 상단 스크롤 진행 바를 통해 남은 분량을 시각적으로 확인하고, 본문을 마크다운 형식이나 정제된 순수 텍스트로 원클릭 복사할 수 있는 내보내기 툴을 제공합니다.
  • 젠 리더(Zen Reader) 지원: 북마크 본문뿐만 아니라, 메모 보드에 작성한 긴 수동 메모 글도 "Zen Reader"를 눌러 동일하게 깨끗한 에세이 형태의 독서 모드로 몰입하여 열람할 수 있습니다.
Q AI 클리너는 어떻게 중복된 북마크를 찾아 정리하나요?

AI 클리너는 전체 북마크 데이터베이스를 스캔하여 URL과 제목의 유사성을 기반으로 중복 항목을 스마트하게 그룹화합니다.

중복이 발견되면 자동으로 가장 먼저 저장된(가장 오래된) 북마크를 유지하도록 권장합니다. 사용자는 그룹을 검토한 후 클릭 한 번으로 불필요한 중복 항목을 안전하게 제거하여 폴더를 깨끗하게 관리할 수 있습니다.

Q ClickBook 버디는 무엇이며 포커스 타이머와 다이얼 메뉴는 어떻게 동작하나요? Premium

ClickBook 버디는 웹페이지 위에 상주하는 귀여운 애니메이션 데스크 펫입니다. 버디를 클릭하면 9가지 빠른 기능을 바로 실행할 수 있는 **원형 다이얼 메뉴**가 열립니다:

  • 번역: 페이지에서 드래그 선택한 텍스트를 대상 언어로 즉시 번역합니다.
  • 북마크: 현재 웹페이지를 클릭 한 번으로 간편하게 북마크에 추가합니다.
  • AI 대화: Gemini Nano 채팅 패널을 열어 페이지 내 실시간 대화를 진행합니다.
  • 타이머: 뽀모도로 집중 타이머를 즉시 시작합니다.
  • 휴식: 편안한 테마 비주얼과 백색소음 사운드가 흐르는 전 화면 휴식 모드를 실행합니다.
  • 메모 & 할 일: 페이지 위에 붙여두는 스티키 메모를 부착하고, 오늘 완료할 체크리스트를 관리합니다.
  • 숨기기 & 설정: 캐릭터 크기, 투명도, 시계 표시 옵션을 세밀하게 조정하며, 현재 도메인에서 숨길 수도 있습니다.

타이머 몰입이 끝나면 **버디 도감**에서 새로운 캐릭터를 수집해 장착할 수 있으며, 오로라 별밤, 고요한 전나무 숲, 명상 요가 등 웅장한 전체화면 인터랙티브 테마로 성공을 축하할 수 있습니다.

Q 마인드맵 뷰에는 어떤 기능들이 있으며 어떻게 조작하나요? Premium

인터랙티브 마인드맵은 전체 폴더 계층과 북마크를 노드 트리 구조의 캔버스로 시각화해 줍니다:

  • 위치 커스터마이징: 마우스 드래그를 통해 각 노드의 배치 위치를 자유롭게 조정할 수 있으며, 이 좌표는 영구적으로 자동 저장됩니다.
  • 비주얼 개인화: 노드 모양(팔각형 등), 테마 HSL 컬러, 연결선 스타일 및 두께 등을 취향대로 바꿀 수 있습니다.
  • AI 서포트 & 일정 동기화: 로컬 AI(Gemini Nano)를 활용해 마인드맵 하위 개념을 자동 확장하고, 해당 노드를 원클릭으로 캘린더 일정이나 TODO 목록으로 직접 내보낼 수 있습니다.
  • 세부 메모 관리: 슬라이드 아웃 사이드바 패널을 통해 마인드맵 각 노드별로 기밀 개인 메모를 별도 기술하고 보관할 수 있습니다.
Q 스프링노트와 스마트 캘린더 기능은 서로 어떻게 연계되나요? Premium

아날로그 스프링노트는 Tiptap 기반의 마크다운 서식 편집 기능과 함께, 이미지·스티키 메모·북마크 카드 등을 자유롭게 배치하는 프리 캔버스를 통합 제공합니다. 캔버스의 모든 리소스들은 글과 연동되어 60fps로 매끄럽게 인라인 스크롤됩니다.

스마트 캘린더는 이 스프링노트, TODO 목록, 메모 보드와 긴밀히 연동됩니다. 드래그앤드롭으로 일정을 편리하게 재배치하거나 주/월/일정 뷰로 확인 가능하며, 한/미/일 공휴일을 실시간 연동해 줍니다.
