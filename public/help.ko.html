소개 및 설치

⚠️ 안내: 본 도움말에는 현재 개발 중인 기능에 대한 설명이 포함되어 있을 수 있습니다. 이에 따라 실제 크롬 웹 스토어 배포 버전과 개발 버전 간에 기능적 차이 및 업데이트 갭이 존재할 수 있음을 알려드립니다.

ClickBook은 Chrome 내장 AI(Gemini Nano)를 활용하여 북마크를 스마트하게 자동 분류하고 관리해 주는 혁신적인 도구입니다. 복잡한 정리 과정 없이 클릭 한 번으로 웹사이트를 저장하고 나중에 쉽게 찾아보세요.

💡 팁: 확장 프로그램 아이콘을 툴바에 고정(Pin)하면 언제 어디서나 즉시 저장할 수 있습니다.

시작하기

1클릭 즉시 저장
웹 서핑 중 저장하고 싶은 페이지에서 ClickBook 아이콘을 클릭하기만 하세요. AI가 분석하여 가장 적절한 폴더에 알아서 넣어줍니다.
단축키 활용
마우스를 움직일 필요도 없습니다. Alt + S 단축키를 눌러 현재 탭을 즉시 저장해 보세요.
관리 대시보드
저장된 북마크는 전용 대시보드에서 한눈에 볼 수 있습니다. 검색, 편집, 이동 등 모든 작업이 여기서 이루어집니다.

AI 지능형 기능 Core

AI 자동 분류
저장 시 사이트 제목과 내용을 분석하여 테크, 디자인, 비즈니스 등 다양한 카테고리로 자동 분류합니다.
AI 자동 정리
이미 저장된 수많은 사이트들을 한꺼번에 재배치하고 싶다면 사이드바의 "AI 자동 정리"를 눌러보세요.
AI 클리너
AI 기반 중복 감지 기능을 통해 중복된 북마크를 찾고 정리할 수 있습니다.
AI 하이라이트 스크랩 Premium
웹페이지 텍스트 드래그 후 우클릭 메뉴로 저장하면, 온디바이스 AI가 사용 언어(한국어/일본어/영어 등)를 분석하여 맞춤 지침 규칙에 맞춰 깨끗한 요약 메모를 자동 생성합니다.
스마트 클리퍼 & 본문 로컬 검색 (FTS) Premium
북마크 저장 시 웹페이지 본문 텍스트만 10ms 이내로 초고속 추출 및 저장합니다. 검색창에서 여러 단어를 공백으로 구분하여 입력하면 본문 전체에서 모든 단어를 포함하는 항목을 찾아내는 다중 단어 실시간 로컬 검색(FTS)을 수행합니다.

생산성 도구

메모 보드 (Memo Board)
북마크마다 중요한 내용을 메모하고, 메모 보드에서 모든 메모를 한꺼번에 모아보세요.
할 일 관리 (Todo Board)
북마크와 연동된 할 일 목록을 작성하여 읽을 거리나 작업을 체계적으로 관리하세요.
태그 보드 (Tag Board)
폴더 구조와 별개로 자유로운 태그를 지정하여 관심사별로 북마크를 필터링할 수 있습니다.
북마크 맵 (Bookmark Map)
내 북마크들의 관계를 시각적인 맵 형태로 한눈에 파악해 보세요.
스마트 절전 모드 Premium
사용하지 않는 유휴 백그라운드 탭을 수면 상태로 전환해 크롬의 RAM 소모량을 최대 90% 아낍니다. 음악 재생이나 고정 탭은 스마트 필터링하며 마우스 호버만으로 즉시 자동 복원됩니다.
오프라인 리더 & 젠 리더 Premium
불필요한 광고나 복잡한 레이아웃을 깨끗하게 소거한 프리미엄 몰입형 독서 화면을 제공합니다. 라이트/세피아/다크 테마, 글꼴 변경, 본문 headings 구조 기반 실시간 목차(TOC)를 내장하고 있으며, 메모용 '젠 리더(Zen Reader)' 모드도 동일하게 지원합니다.
스마트 캘린더 연동 & 관리 Premium
할 일(Todo) 목록과 유기적으로 동기화되는 스마트 캘린더를 제공합니다. Nager.Date API를 활용한 국가별(한국/미국/일본 등) 공휴일 실시간 자동 동기화 및 캐싱을 지원하며, 월간/주간/일간 뷰 전환, HTML5 드래그 앤 드롭을 통한 마우스 드래그 일정 및 시간 변경, 매일/매주/매월/매년 반복 일정 등록을 지원하여 일정을 완벽히 체계화합니다.

GitHub Trending

Hugging Face 랭킹

폴더 및 북마크 관리

  • 무한 계층: 폴더 안에 폴더를 무제한으로 만들어 체계적으로 관리할 수 있습니다.
  • 드래그 앤 드롭: 사이트나 폴더를 원하는 곳으로 끌어다 놓으세요. 상위 폴더 사이로 드래그하면 같은 층위로 이동합니다.
  • 폴더 잠금: AI 자동 정리 시 해당 폴더의 내용이 바뀌지 않도록 잠글 수 있습니다.
  • 커스텀 아이콘: 각 폴더에 개성 있는 이모지나 아이콘을 설정하여 시각적으로 꾸며보세요.
  • 탭 그룹 양방향 동기화 Premium: 크롬 순정 탭 그룹을 클릭북 폴더로 한 번에 백업하고, 반대로 폴더 메뉴를 통해 고유 색상과 이름을 유지한 채 순정 크롬 탭 그룹으로 즉시 재생성 복원합니다.

Chrome 북마크 연동

북마크 가져오기
기존에 사용하던 Chrome 북마크를 ClickBook으로 즉시 가져옵니다. 가져오는 동안 AI를 통해 자동 분류할 수도 있습니다.
Chrome으로 내보내기
ClickBook에서 정리한 폴더 구조를 Chrome 순정 북마크 매니저로 다시 내보내어 다른 기기와 동기화할 수 있습니다.

설정 및 데이터 관리

  • 테마 설정: 라이트, 다크 또는 시스템 설정에 맞춘 테마를 자유롭게 선택할 수 있습니다.
  • 백업 및 복구: 전체 데이터를 JSON 파일로 내보내 안전하게 보관하고, 필요할 때 언제든 복구하세요.
  • 데이터 초기화: 모든 로컬 데이터를 삭제하고 새로 시작할 수 있습니다. (주의: 삭제된 데이터는 복구할 수 없습니다.)

프라이버시 및 보안

100% 로컬 작동
ClickBook은 어떤 외부 서버와도 통신하지 않습니다. 데이터는 오직 귀하의 컴퓨터 내부(Local Storage)에만 저장됩니다.
보안 폴더 및 자동 세션 파쇄기 (Secure Folder) Premium
기밀 정보나 금융 페이지 폴더를 '보안 폴더'로 지정해 보세요.
  • 실시간 감지 배지: 보안 사이트에 진입하면 크롬 툴바 아이콘 위에 초록색 `SEC` 배지가 동적으로 점등되어 보호 중임을 알립니다.
  • 진입 알림 토스트: 페이지 로딩 완료 시 우측 하단에 우아한 보안 세션 가동 알림창이 1회 표기되어 안정감을 제공합니다.
  • 종료 시 자동 파쇄: 탐색을 마친 뒤 탭을 닫으면, 관련 쿠키, 로컬스토리지, 캐시, IndexedDB 및 방문 흔적을 완전 자동 소거하여 흔적 없는 완벽한 개인정보 보호를 달성합니다.
