# ClickBook AI 기능 확장 아이디어 (Chrome Gemini Nano 활용)

현재 ClickBook은 Chrome 내장 AI(Gemini Nano)를 활용하여 **자동 분류, 검색어 확장, 추천 사이트 제안, 전체 재배치** 등의 훌륭한 기능들을 제공하고 있습니다. 로컬 AI의 장점(빠른 속도, 프라이버시 보호, 무료 제공)을 극대화하여 ClickBook에 추가할 수 있는 잠재적인 기능 아이디어들을 정리합니다.

## 1. 📖 AI 스마트 요약 (TL;DR) & 태그 자동 생성
단순한 카테고리 분류를 넘어, 사이트의 핵심 내용을 요약하고 태그를 부여합니다.

*   **기능 상세:**
    *   북마크를 저장할 때 (또는 우측 패널에서 상세 정보를 열 때) AI가 사이트 제목과 URL 등을 바탕으로 1~2줄의 짧은 요약(TL;DR)을 생성합니다.
    *   `#react`, `#tutorial`, `#finance` 등 직관적인 해시태그 3~5개를 자동으로 추출합니다.
*   **기대 효과:** 시간이 지난 뒤 북마크를 다시 열어볼 때 해당 사이트를 왜 저장했는지 쉽게 파악할 수 있으며, 향후 태그 기반 검색으로 발전시킬 수 있습니다.
*   **구현 방향:** `prompt` 요청 시 카테고리뿐 아니라 `{"summary": "...", "tags": [...]}` 형태의 JSON 응답을 요구하도록 프롬프트를 설계합니다.

## 2. 🔍 자연어 기반 의도 파악 필터링
단어 매칭 검색을 넘어 사용자의 문장(자연어) 의도를 파악하여 고급 검색을 수행합니다.

*   **기능 상세:**
    *   사용자가 검색창에 "최근에 저장한 리액트 관련 사이트"라고 입력합니다.
    *   AI가 이 자연어 문장을 분석하여 앱이 이해할 수 있는 필터 객체(예: `{ "keyword": "react", "timeframe": "recent", "category": null }`)로 변환합니다.
*   **기대 효과:** 복잡한 필터 버튼(날짜, 카테고리 등)을 누를 필요 없이 검색창 하나로 스마트하게 필터링이 가능합니다.
*   **구현 방향:** 자연어를 입력받아 정해진 스키마의 JSON 필터 조건으로 반환하는 백그라운드 AI 세션을 구축합니다.

## 3. 🧹 의미론적 중복/유사 북마크 정리 (AI 클리너)
URL이 일치하는 단순 중복 검사를 넘어, '같은 주제를 다루는 문서'들을 찾아내어 정리를 돕습니다.

*   **기능 상세:**
    *   '북마크 정리' 버튼을 누르면 AI가 전체 북마크 리스트를 분석합니다.
    *   서로 다른 URL이지만 내용은 같은 부류(예: "React Router v6 튜토리얼" 관련 블로그 글 3개)를 찾아내어 그룹화합니다.
    *   사용자에게 "이 북마크들은 주제가 겹칩니다. 하나만 남기시겠습니까?" 또는 "단일 폴더로 묶으시겠습니까?"라고 제안합니다.
*   **기대 효과:** 정보의 파편화를 막고 방치되는 북마크를 줄여 앱의 효용성을 높입니다.
*   **구현 방향:** 배치(batch) 단위로 북마크 목록을 AI에게 주고 의미상 유사한 항목들의 ID를 배열로 묶어 반환하도록 합니다.

## 4. 💡 "오늘의 발견" (AI 북마크 큐레이션)
쌓여있는 북마크를 다시 소비하도록 유도하는 리마인드 기능입니다.

*   **기능 상세:**
    *   대시보드 상단에 **[오늘의 AI 픽]** 또는 **[잊혀진 북마크]** 섹션을 제공합니다.
    *   방문 횟수(visitCount)가 0이거나 저장된 지 오래된 북마크 중, 현재 사용자의 최근 관심사(최근 저장 패턴)와 연관이 있는 항목을 3~4개 선별하여 추천합니다.
*   **기대 효과:** 북마크 앱이 단순히 '저장소'에 머무르지 않고, 지식을 다시 일깨워주는 '비서' 역할을 하게 됩니다.
*   **구현 방향:** 로컬 브라우저의 저장된 데이터를 바탕으로 추천 알고리즘을 돌리거나, AI에게 랜덤 샘플링된 목록을 주고 가장 흥미로운 3개를 뽑아 이유와 함께 설명하게 합니다.

## 5. 📝 메모 보드 자동 초안 작성 (AI Memo)
사용자가 직접 기록하는 '메모 보드'의 허들을 낮춰줍니다.

*   **기능 상세:**
    *   특정 북마크에 대해 '메모 쓰기'를 클릭하면 빈 텍스트 박스가 나오는 대신, AI가 미리 짧은 초안을 작성해 둡니다.
    *   예: "이 사이트는 〇〇을 다루고 있습니다. 향후 〇〇 프로젝트에 참고하기 위해 저장함."
    *   사용자는 이 초안을 그대로 저장하거나 조금만 덧붙여서 기록합니다.
*   **기대 효과:** 메모 작성이 훨씬 쉬워져 사용자 참여도와 데이터의 풍부함이 증가합니다.
*   **구현 방향:** 메모 모달을 띄울 때 비동기로 AI 요약을 호출하여 textarea의 기본값(defaultValue)으로 넣어줍니다.

---
**기술적 고려사항 (Chrome Gemini Nano)**
*   **토큰 및 시간 제한:** 한 번에 너무 많은 북마크를 넘기면 컨텍스트 윈도우를 초과하거나 타임아웃이 발생할 수 있으므로, 현재 'AI Reorganize' 기능처럼 배치(Batch) 처리 방식을 유지해야 합니다.
*   **Fallback 로직:** AI 세션 생성이 실패하거나 답변이 지연될 경우를 대비해, UI 진행률 표시기를 명확히 하고 언제든 중단(Abort) 가능하도록 설계해야 합니다.
