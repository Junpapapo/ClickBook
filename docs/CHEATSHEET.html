Phase 3.5 — Gemini Nano（Chrome AI）の有効化

Gemini Nano とは： Chrome に内蔵されたオンデバイス AI モデル。外部サーバーへの通信なし・API キー不要でローカル推論が可能。ClickBook は window.ai.languageModel を通じてアクセスし、利用不可の場合はルールベース分類に自動フォールバックする。
1
Chrome Canary を使用する（推奨）
Gemini Nano は現在 Chrome Canary（実験的ビルド）での利用を推奨。通常版 Chrome でも有効化できる場合がある。
https://www.google.com/chrome/canary/
2
フラグ ① — オンデバイスモデルを有効化
chrome://flags/#optimization-guide-on-device-model
Enabled BypassPerfRequirement に設定 → Chrome を再起動
3
フラグ ② — Prompt API を有効化
chrome://flags/#prompt-api-for-gemini-nano
Enabled に設定 → Chrome を再起動
4
モデルのダウンロード確認
chrome://components
「Optimization Guide On Device Model」のバージョンが 0.0.0.0 の場合は「アップデートを確認」をクリック。モデルは数百 MB あるためダウンロードに時間がかかることがある。
5
DevTools で利用可否を確認
任意のページで F12 → Console タブに以下を貼り付けて実行：
(await window.ai.languageModel.capabilities()).available
"readily" → Nano 利用可能（ClickBook の AI ボタンが有効になる）
"after-download" → モデルダウンロード中。しばらく待つ
"no" → フラグが未設定または非対応バージョン
🤖 AI 自動整理 のテスト NEW
  • 管理ページを開き、複数のブックマークが登録されていることを確認
  • 左サイドバーの「AI自動整理」ボタン（紫グラデーション ✦ アイコン）が有効になっていることを確認
  • ボタンをクリック → 確認ダイアログが表示される
  • 「AI整理を実行」をクリック → スピナーが表示され処理が始まる
  • 完了後、サイドバーに 「✓ AI整理完了 〇件中△件を移動しました」 が 6 秒間表示される
  • パターン一覧（PatternBar）に AI整理バックアップ YYYY/MM/DD HH:mm が自動保存されていることを確認
  • パターンを復元するとバックアップ時点の状態に戻せることを確認
Nano 利用不可の場合： 「AI自動整理」ボタンは自動的に無効化され「利用不可」バッジが表示される。上記フラグを有効にした後、Chrome と拡張機能を再読み込みすること。

開発コマンド集

🔨 ビルド（本番）
npm run build

dist/ を生成。Chrome に読み込む際はこちら。

👀 ウォッチビルド（開発）
npm run dev

ファイル保存のたびに自動ビルド。Chrome で 🔄 更新が必要。

✅ TypeScript チェック
npx tsc --noEmit

型エラーのみ確認。ビルドは行わない。

🧹 dist クリーン & 再ビルド
rm -rf dist && npm run build

キャッシュ起因の問題が疑われる場合に使用。

デバッグ方法

🐛 Service Worker のログ確認
Service Worker の console.log はここで確認する
  • chrome://extensions を開く
  • ClickBook カードの「Service Worker」リンクをクリック
  • DevTools の Console タブで確認
🔍 Popup の DevTools
  • Popup を右クリック → 「検証」
  • または Popup を開いた状態で F12
💾 ストレージの確認
  • 管理ページ（newtab）で F12 → Application タブ
  • Storage → Extension storage → Local Storage
  • キー clickbook_data を確認
🗑️ ストレージのリセット
  • DevTools Console で以下を実行：
chrome.storage.local.clear()

再度ページを開くとデフォルト8フォルダーが復元される

拡張機能を更新した後： chrome://extensions の ClickBook の 🔄 ボタンをクリックし、管理ページのタブも閉じて再度開くこと。
